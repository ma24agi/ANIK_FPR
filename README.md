This work presents a system that employs a Convolutional Neural Network (CNN) approach for the classification of the type of Brain Tumor using sophisticated techniques of Deep Learning and Explainable AI-XAI. The system utilizes medical images based on four labels, including Glioma, Meningioma, Pituitary, and No Tumor. It has been developed and tested using the PyTorch library. The model efficiently uses state-of-the-art CNNs like EfficientNet_B0, MobileNet, and ResNet50. The model has been trained and validated on a dataset of around 3,000 images. The results show outstanding performance. The model with the best performance, EfficientNet_B0, resulted in training accuracy and validation accuracy of 99.04% and 99.29%, with training loss and validation loss of 0.0322 and 0.0247. Techniques from explainable AI were implemented to improve transparency and interpretability, and insights were provided regarding the decision-making processes of these models. There is an integration of XAI in this system to ensure its reliability and usability in real medical applications.
